Creacion del html

Se estructura el encabezado (header) y el pie de pagina (footer) y se dispone todo el contenido dentro del cuerpo en un main

  • Dentro del header se crean un elemnto h1 donde va el titulo de la pagina
  • Se crea un parrafo donde se colocara el titulo y la descripcion del titulo

Dentro del main se disponen 2 secciones () principales donde se colocaran respectivamente el contenido de los cafe y los postres.

En cada seccion se crea un encabezado () donde se colocara el titulo y la imagen de referencia para la seccion.

Se crean 5 articulos (articles) en cada seccion en los cuales iran posteriormente los contenidos de cada uno

Dentro de cada articulos (articles) se crean 2 parrafos (p) y se coloca en cada uno de ellos la informacion correspondiente:

  • Nombre del producto
  • Precio del producto
